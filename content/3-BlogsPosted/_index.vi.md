Tại đây sẽ là phần liệt kê, giới thiệu các bài viết blog mà bạn đã đăng trên [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj).

### [Blog 1 - Kinh Nghiệm Thực Chiến: Giải Bài Toán Đọc Hợp Đồng Tự Động Với Doczy.ai Trên AWS](3.1-Blog1/)
Bài viết chia sẻ case study thực tế về hệ thống Doczy.ai™ do AArete phát triển, giải quyết triệt để bài toán tự động bóc tách dữ liệu phi cấu trúc từ hợp đồng pháp lý bằng Generative AI (Amazon Bedrock) và Amazon Textract trên nền tảng AWS.

### [Blog 2 - Tự động hóa số hóa hồ sơ y tế với Amazon Bedrock Data Automation & AWS HealthLake](3.2-Blog2/)
Bài viết trình bày phương pháp kết hợp Generative AI (Amazon Bedrock Data Automation) và AWS HealthLake để tự động trích xuất, số hóa và lưu trữ đồng bộ hồ sơ bệnh án không cấu trúc theo chuẩn dữ liệu y tế quốc tế FHIR.

### [Blog 3 - Tự động hóa và mở rộng quy mô di chuyển AWS Storage Gateway sang AL2023 bằng Infrastructure as Code](3.3-Blog3/)
Bài viết trình bày phương pháp tự động hóa quá trình di chuyển thiết bị AWS Storage Gateway từ Amazon Linux 2 sang Amazon Linux 2023 ở quy mô lớn bằng cách sử dụng kết hợp Terraform và Ansible.
